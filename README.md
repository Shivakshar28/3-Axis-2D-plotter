# 3-Axis 2D Plotter Machine | SolidWorks

## Overview

This repository contains the complete CAD design, assembly, and documentation of a **3-Axis 2D Plotter Machine** developed using **SolidWorks**. The project demonstrates the design and integration of a three-axis motion system capable of accurately drawing 2D shapes, patterns, and text by controlling movement along the X, Y, and Z axes.

In addition to the CAD design, a **fully functional hardware prototype** of the plotter machine was successfully fabricated and tested, validating the design and demonstrating its practical implementation.

## Project Objectives

The primary objective of this project was to design, model, and build a cost effective and precise 2D plotting machine while gaining hands-on experience in mechanical design, CAD modeling, prototyping, and system integration. The project emphasizes the relationship between digital design and physical manufacturing by taking the concept from a SolidWorks model to a working machine.

## Features

* Complete 3D CAD model designed in SolidWorks
* Parametric modeling of all mechanical components
* Fully assembled machine with proper mechanical constraints
* Functional X, Y, and Z axis movement mechanism
* Motion validation through assembly design
* Fabricated and tested working prototype

## Working Prototype

A physical working model of the 3-Axis 2D Plotter Machine was built based on the SolidWorks design. The prototype successfully performs controlled movement along the X, Y, and Z axes to draw 2D figures and text with high positional accuracy. The successful implementation demonstrates the feasibility of the design and validates the mechanical assembly and motion mechanism.
